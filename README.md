# Short Form Video Script Writer

An AI skill for writing short-form video scripts (TikTok, Instagram Reels, YouTube Shorts) that are built to perform — not just summarize a product or concept.

The output is **spoken-word only** — no stage directions, no storyboard labels. Ready to hand directly to a presenter or paste into a production tool.

---

## What It Does

- Generates topic and angle ideas before writing (the angle is what gets watched)
- Calibrates voice and persona to the presenter — including AI avatar-specific adjustments
- Chooses from 6 proven script frameworks (Hook-Value-CTA, PAS, BAB, PSP, AIDA, PASTOR)
- Constructs high-retention hooks (Bold Statement, Question, Pattern Interrupt, Proof-First)
- Adds a re-hook at the 5–7 second mark to prevent mid-video drop-off
- Builds body copy using open loop retention techniques
- Applies psychological triggers: identity alignment, contrast pairs, rule of three, FOMO, specificity as authority
- Handles series architecture — fixed premise, variable subjects, standalone episode rule
- Writes platform-adapted captions for TikTok, Reels, and Shorts
- Runs a 10-point revision checklist before output
- Provides a post-performance analysis loop to improve every next script

---

## Default Behavior

One cross-platform script, **45–60 seconds**, optimized to perform on TikTok, Instagram Reels, and YouTube Shorts simultaneously. The words are identical across all three. Captions, hashtags, and posting behavior adapt per platform.

Platform-specific mode (e.g., 15-second Reel or 90-second YouTube Short) available on request.
