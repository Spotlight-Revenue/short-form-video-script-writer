# Short-Form Video Script — Quick Reference

## Frameworks at a Glance

| Framework | Best for |
|-----------|---------|
| Hook-Value-CTA | Beginners, educational |
| PAS | Pain-point content, aware audience |
| BAB | Transformation, case studies |
| PSP | Algorithm-optimized, builder/peer |
| AIDA | Classic marketing arc |
| PASTOR | Trust-building, 60–90 sec, founder style |

---

## Hook Types

| Type | When to use | Example |
|------|------------|---------|
| **Bold Statement** | Challenge a belief | "Your agent can run your entire ClickUp workspace." |
| **Question** | Open an information void | "Why do some scripts get 1M views while others die at 500?" |
| **Pattern Interrupt** | Hijack attention | Unexpected opener that still connects to content |
| **Proof-First** | Show result, tease method | "I built a skill that took over full ClickUp management in one setup." |

---

## Runtime Targets

| Runtime | Word count |
|---------|-----------|
| 30 sec | 60–75 words |
| 45 sec | 90–110 words |
| 60 sec | 125–150 words (cross-platform sweet spot) |
| 90 sec | 185–225 words |

Speaking pace: ~2 words/second. Always read aloud and time it.

---

## Psychological Triggers Cheat Sheet

- **Identity alignment** — "If you're building X, this is worth your time."
- **FOMO framing** — "It's on ClawHub." (others are already using it)
- **Contrast pair** — "Not access. Operation."
- **Rule of three** — "Route them, assign them, date them."
- **Specificity as authority** — Specific details imply real knowledge
- **"But" pivot** — Concession before payoff builds credibility

---

## Audience Depth Spectrum

| Level | Hook style |
|-------|-----------|
| Beginner | Pain point, frustration, surprising stat |
| Aware | Problem agitation + reveal |
| Informed | Differentiation hook |
| Builder/Peer | Peer share, bold claim, "wait, how?" |

**Builder/peer rule:** Never explain what the tool is. Talk like a peer.

---

## Caption Rules (All Platforms)

- Never start with a hashtag
- Don't transcribe the video
- Make the caption a second hook for silent viewers

| Platform | Length | Focus |
|----------|--------|-------|
| TikTok | Up to 4,000 chars | Keyword-rich, hook line first |
| Reels | 100–300 chars | One punchy line that adds a layer |
| Shorts | Keyword-rich title | Write title like a search query |

---

## 10-Point Revision Checklist

- [ ] Hook opens a loop
- [ ] Re-hook at 5–7 second mark
- [ ] One idea only
- [ ] At least one psychological trigger in the body
- [ ] Read aloud — sounds natural?
- [ ] Word count within target
- [ ] Character count within production limit
- [ ] Loop closed at same intensity as hook
- [ ] CTA flows naturally
- [ ] Voice match — sounds like the presenter?

---

## Performance Benchmarks

| Metric | Good | Needs work |
|--------|------|-----------|
| 3-sec hold rate | 65%+ | Below 40% = rewrite hook |
| Completion rate | High | Drop spike = body pacing issue |
| Save rate | High | Low = value problem |
| "When's the next one?" comments | Present | Best signal you can get |

**Iteration rule:** Run 3–5 videos, then debrief and adjust. Don't iterate one video at a time.
